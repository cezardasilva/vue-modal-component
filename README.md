> A simple modal component for Vue.js 2.x

# Instalation
```bash
npm install vue-modal-component --save-dev
```

# Usage
```Vue
<template>

</template>

<script>

import Modal from 'vue-modal-component'
export default {
	name: "YourComponent",
	components: {
	    Modal
	}
}
</script>
```
